# 公益表单系统

#### Description
一个开源开放的表单系统，可以用于收集机构内部的隐私数据，并发布或者不发布到公开场合；
也可以发布到公开场合收集数据。
也可以作为上榜打开的数据收集表单。
还可以作为信息同步工具。
